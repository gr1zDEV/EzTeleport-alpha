## What EzTeleport does

EzTeleport lets you define custom teleport-style commands (for example `/hub`, `/spawn`, `/afk`) with configurable:

- Countdown
- Cooldown
- Cancel-on-move and cancel-on-damage behavior
- Destination (fixed coordinates *or* command execution)
- Chat/action bar messages and sounds
